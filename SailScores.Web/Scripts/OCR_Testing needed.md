# OCR Race Entry Testing Checklist

## Testing Checklist
- [ ] Compatible with speech recognition

- [ ] O ? 0 (letter O to zero)
- [ ] l/I ? 1 (lowercase L or capital I to one)
- [ ] S ? 5 (letter S to five)
- [ ] Exact sail number match (100% confidence)
- [ ] Alternative sail number match (95% confidence)
- [ ] Fuzzy match (ignoring spaces/dashes, 85% confidence)
